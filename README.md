# 🛒 E-Commerce Sales Analysis Dashboard (Excel)

##  Project Overview
This project analyzes an E-commerce sales dataset using **Microsoft Excel** and **Pivot Tables** to generate business insights.

The dashboard focuses on profitability, customer behavior, discount impact, and transaction trends across different categories and store types.

---

##  Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- KPI Cards (Excel-based calculations)

---

##  Dashboard Insights

### 🔹 Store Type vs Profit
Analyzed which store types generate higher profitability.

### 🔹 Loyalty Level vs Customer Count
Identified customer distribution based on loyalty membership levels.

### 🔹 Year vs Discount
Examined how discount trends change year over year.

### 🔹 Year vs Profit & Transactions
Compared annual profit performance and transaction volume.

### 🔹 Category vs Profit
Evaluated which product categories contribute most to profit.

### 🔹 KPI Cards Included
- Total Profit  
- Total Transactions  
- Total Customers  
- Average Discount  
- Profit Growth  

---

##  Filters Used
